Instructions for Installing and Running AuthBillPro

1. Run the installation script:
   - For Windows, double-click `install.bat` or right-click and select `Run as Administrator`.

2. Follow the on-screen instructions to complete the installation.

3. Once the installation is complete, open `AuthBillPro.accdb` or `AuthBillPro.accde` using Microsoft Access Runtime.

If you encounter any issues, please contact support.
